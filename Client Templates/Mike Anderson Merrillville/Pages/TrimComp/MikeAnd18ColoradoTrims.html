2018 Chevrolet Colorado Base

Starting MSRP: $21,195

Choose the 2018 Chevy Colorado Base for an affordable yet capable drive. You can expect the following features:

  • 5L I4 DOHC VVT engine
  • Extended cab and long box configuration only
  • StabiliTrak® electronic stability control system
  • Available in 2WD only
  • 16-in. ultra silver metallic steel wheels
  • Cargo area lamps
  • Daytime running lights and halogen projector-beam headlamps
  • 5-in. diagonal monochromatic driver information center
  • 6-speaker audio system
  • Chevrolet MyLink® Radio with 7-in. touchscreen
  • Rear vision camera
  • Tire pressure monitoring system
