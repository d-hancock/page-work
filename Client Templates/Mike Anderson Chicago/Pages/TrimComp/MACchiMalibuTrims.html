2018 Chevrolet Malibu L

Starting MSRP: $22,555

This affordable base level Malibu trim has quite a few features you’d expect on a more expensive model, including the following:

  • 5L turbocharged 4-cylinder engine
  • 6-speed automatic transmission
  • Daytime running lamps
  • 16-in. steel wheels with cover
  • 6-speaker sound system
  • Carpeted floor mats
  • Power windows

2018 Chevrolet Malibu LS

Starting MSRP: $24,195

The Chevy Malibu LS comes with a number of impressive features that you’re sure to take advantage of. The Malibu LS includes many Malibu L features, plus:

  • 16-in. aluminum wheels
  • Chverolet MyLink® Radio with 7-in. diagonal color touchscreen
  • Compass display
  • Built-in 4G LTE Wi-Fi® hotspot
  • Rear vision camera
2018 Malibu LS Details

2018 Chevrolet Malibu LT

Starting MSRP: $26,095

Expect even more excellent features with this mid-level Chevrolet Malibu model. The Malibu LT includes many Malibu LS features, plus:

  • LED daytime running lamps
  • Outside heated power-adjustable integrated turn signal mirrors
  • 17-in. aluminum wheels
  • SiriusXM® Satellite Radio
2018 Malibu LT Details

2018 Chevrolet Malibu Premier

Starting MSRP: $31,895

As the top-of-the-line trim option, the 2018 Chevrolet Malibu Premier is packed with a ton of impressive options to make your daily drive exceptional. The Malibu Premier includes many Malibu LT features, plus:

  • 0L turbocharged 4-cylinder engine
  • 18-in. aluminum wheels
  • Chevrolet MyLink® with 8-in. diagonal color touchscreen and navigation
  • Bose® premium 9-speaker sound system
  • Dual-zone automatic climate control
  • Heated front seats
  • Heated and leather-wrapped steering wheel
  • 2-in color driver information center
  • Leather-wrapped shift knob
2018 Malibu Premier Details

2018 Chevrolet Malibu Hybrid

Starting MSRP: $28,795

The Malibu Hybrid comes with a number of Malibu LT features, but instead of a gasoline-powered engine, it’s equipped with a 1.8L hybrid 4-cylinder engine, as well as an automatic, two-motor drive unit hybrid transmission.
